## R/Pharma Hangouts 

Welcome to the [R/Pharma](https://rinpharma.com) Hangouts repository! The R/Pharma Hangout sessions are a monthly event where we feature key innovators in the world of open source to go hands-on with a key R package, application, or entire workflow that is highly relevant to the life sciences industry. You can find additional background and motivation for these sessions on the [R/Pharma Hangouts blog post](https://rinpharma.com/docs/blog/posts/2026-02-09-hangouts/). This repository contains example projects demonstrated during these sessions.
